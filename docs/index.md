# TrendPublish 文档

TrendPublish 是一个基于 Deno 的趋势发现和内容发布系统，支持多源采集、AI 处理与微信公众号自动发布。

## 功能清单

- 多源内容抓取（Twitter/X、FireCrawl、Jina）
- 内容筛选、摘要与标题生成
- 微信公众号图文渲染与发布
- 定时任务与手动触发并存
- Bark / 钉钉 / 飞书通知
